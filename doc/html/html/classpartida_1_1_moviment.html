Detailed Description

Un moviment amb coordenades i atributs de matar i saltar.

Les coordenades de moviment es calculen de forma horitzontal i vertical, de forma que ens indiquen si son moviments definits o lineals. L'atribut movMatar ens indica si el moviment permet matar Peces; l'atribut movSlaltar ens indica si el moviment permet saltar Peces.
